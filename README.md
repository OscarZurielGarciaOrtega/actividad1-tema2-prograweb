Actividad 1 - Tema 2: Ejercicios de Programación Web
Este repositorio contiene la colección de ejercicios prácticos correspondientes a la Actividad 1 del Tema 2 de la materia de Programación Web.

Objetivo de la Actividad
Demostrar el dominio práctico en el desarrollo de interfaces web utilizando HTML5, CSS3 y JavaScript, estructurando correctamente los documentos y manejando eventos del DOM de forma dinámica (ej. calculadoras y conversores de datos).

Contenido del Repositorio
index.html: Menú principal de navegación.

ejercicio1.html al ejercicio10.html: Código fuente de los ejercicios resueltos.

/css/, /img/ y /js/: Directorios con las hojas de estilo, imágenes y scripts correspondientes.

Despliegue en Línea
El proyecto completo está alojado y funcionando mediante GitHub Pages, permitiendo la interacción con los formularios y el código JavaScript en un entorno real.
